# Quantum Error Correction

A series of articles introducing quantum error correction (QEC) from the ground up.

## Articles

1. **Introduction** — motivation and scope
2. **Classical Error Correction** — the repetition code, code distance, and logical error rates
3. **Why QEC is Hard** — no-cloning, continuous errors, and destructive measurement
4. **Adding Redundancy** — entanglement as redundancy, the codespace, and error subspaces
5. **Bit Flip Errors** — parity-checking circuits and syndrome extraction
6. **Phase Errors** — the Hadamard transform and phase-flip detection
7. **Shor's Code** — combining bit and phase flip correction
8. **Digitisation of Errors** — why correcting X and Z is enough for arbitrary errors
9. **Simulating with Stim** — hands-on simulation of Shor's code
10. Many More coming
